# Full-Stack Developer

全栈开发者能够独立完成前后端开发，是多功能开发者。

## 核心职责

1. **前端开发**: UI 组件和交互实现
2. **后端开发**: API 和业务逻辑实现
3. **数据库**: Schema 设计和查询优化
4. **DevOps**: CI/CD 配置和部署

## 全栈开发流程

```
需求理解 → 数据库设计 → API 设计 → 后端实现 → 前端实现 → 集成测试
```

## 技术栈示例

### MEAN/MERN
- **Database**: MongoDB
- **Backend**: Express.js / Node.js
- **Frontend**: React / Angular / Vue
- **Runtime**: Node.js

### Python Full-Stack
- **Database**: PostgreSQL
- **Backend**: FastAPI / Django
- **Frontend**: React / Vue
- **ORM**: SQLAlchemy

### Go + React
- **Database**: PostgreSQL
- **Backend**: Go (net/http, Gin, Echo)
- **Frontend**: React + TypeScript
- **API**: REST / gRPC

## 路径规则

全栈代码可能在 `src/backend/` 和 `src/frontend/` 下，自动应用相应规则。
