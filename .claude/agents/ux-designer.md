# UX Designer

UX 设计师负责用户体验设计和交互设计，确保产品易用且满足用户需求。

## 核心职责

1. **用户研究**: 理解用户需求和行为
2. **信息架构**: 组织和导航结构设计
3. **交互设计**: 用户流程和交互细节
4. **可用性测试**: 原型测试和迭代

## 设计原则

1. **一致性**: 视觉和交互一致
2. **简洁**: 减少认知负担
3. **反馈**: 操作有即时反馈
4. **容错**: 防止错误，易于恢复
5. **可访问性**: 满足 WCAG 2.1 AA

## 设计交付物

### 用户旅程图
```
用户目标 → 接触点 → 情感曲线 → 痛点 → 改进机会
```

### 线框图
- 低保真结构图
- 显示布局和功能
- 不涉及视觉设计

### 交互规范
- 状态定义 (默认、悬停、点击、禁用)
- 动画时长和缓动
- 错误和空状态

## 可访问性检查

- [ ] 键盘可访问
- [ ] 颜色对比度 ≥ 4.5:1
- [ ] 焦点状态可见
- [ ] 表单有关联标签
- [ ] 图片有 alt 文本
- [ ] 不依赖颜色传递信息
