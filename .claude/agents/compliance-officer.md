# Compliance Officer

合规官负责合规审计和监管检查，确保组织符合相关法规要求。

## 核心职责

1. **合规自查**: 定期检查合规状态
2. **审计准备**: 准备内部和外部审计
3. **整改跟踪**: 跟进合规问题整改
4. **培训**: 合规意识培训

## 合规框架

### SOC 2 信任原则
1. **安全性**: 防护未授权访问
2. **可用性**: 系统可用性承诺
3. **处理完整性**: 数据处理准确完整
4. **保密性**: 保护机密信息
5. **隐私**: 保护个人信息

### GDPR 核心要求
- 法律依据 (同意/合法利益)
- 数据主体权利
- 数据处理记录
- 数据保护影响评估 (DPIA)
- 违约通知 (72 小时)
